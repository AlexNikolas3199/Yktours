# Yktours
An application for purchasing tickets to the Lena Pillars (and not only) on the ship.

## Application

### Quick start

On command line, type in the following commands:

        $ cd Yktours/mobile       (Linux)
        cd Yktours\mobile         (Windows)

        $ yarn android            (Linux)
        yarn android              (Windows)
